## 📖 Graduation Roadmap & Project Scopes

### Day 85 - 87: Developing a Capstone Project
For your Capstone project, choose one of the following templates or pitch your own idea:
1.  **Full-Stack Blog Engine (Web/DB)**:
    *   An interactive FastAPI web portal connected to a SQLite/PostgreSQL database.
    *   Supports user authentication, password hashing, posting, commenting, and sorting posts by category tags.
    *   100% test coverage using `pytest`.
2.  **Personal Finance Dashboard (Data Science/APIs)**:
    *   A program that automatically imports bank statement logs, queries live stock prices from a financial market API, and calculates current net worth.
    *   Generates analytical Pandas data calculations and visualizes trends using Seaborn plots.
3.  **Automatic Job Finder & Scraper (Automation/CLI)**:
    *   A script that searches job recruitment boards (e.g., Hacker News Jobs) for specific tech keywords.
    *   Saves search outputs, automatically generates email applications, and logs all applications to a persistent database.

## 📖 Daily Lessons

### Day 8: Conditional Statements
Conditionals execute blocks of code depending on whether an expression evaluates to `True` or `False`.
```python
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C")
```
*Key Rule*: Pay close attention to **indentation** (4 spaces). Python uses spaces, not curly braces, to define code scopes!

---

### Day 9: Logical Operators & Nested Conditionals
Combine multiple checks in a single conditional statement:
*   `and`: Returns `True` only if *both* sides are true.
*   `or`: Returns `True` if *at least one* side is true.
*   `not`: Reverses a boolean value (`not True` -> `False`).

```python
has_ticket = True
has_id = False
if has_ticket and has_id:
    print("Access Granted")
else:
    print("Access Denied - Requires both Ticket and ID.")
```

### Day 10: Iteration with `while` Loops
A `while` loop runs code repeatedly *as long as* a specified condition remains `True`. Always ensure the condition eventually becomes `False` to avoid infinite loops!
```python
count = 1
while count <= 5:
    print(f"Counting: {count}")
    count += 1
```

---

### Day 11: Iteration with `for` Loops
A `for` loop is used to iterate over a sequence (such as a string, list, or number series generated by `range()`).
```python
# Print numbers from 0 to 4
for i in range(5):
    print(i)
    
# range(start, stop, step)
for i in range(2, 10, 2):
    print(i) # Prints 2, 4, 6, 8
```

---

### Day 12: Loop Controls: break, continue, and pass
Fine-tune your loop iteration behaviors:
*   `break`: Exits the loop immediately.
*   `continue`: Skips the rest of the current iteration and jumps directly to the next loop pass.
*   `pass`: A null statement used as a placeholder.

```python
for num in range(1, 6):
    if num == 3:
        continue # Skip printing 3
    print(num)
```

### Day 13: Nested Loops & Ternary Operators
*   **Nested Loops**: A loop running inside another loop (e.g., coordinate grids).
*   **Ternary Operator**: Inline compact conditionals.
    ```python
    age = 18
    status = "Adult" if age >= 18 else "Minor"
    print(status)
    ```
